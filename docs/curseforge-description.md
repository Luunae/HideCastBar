# Hide CastBar

Hides the default player casting bar in World of Warcraft Mists of Pandaria Classic (5.5.x).

Nothing else.

## Slash commands

| Command | What it does |
|---|---|
| `/hcb` | Toggle the castbar between hidden and shown |
| `/hidecastbar` | Same as `/hcb` |

State persists across sessions (saved per account). The castbar is hidden by default on first install.

## Compatibility

Targets MoP Classic 5.5.x. I'm not opposed to expanding to other client versions (Classic Era, Wrath Classic, retail, etc.), but I don't have a reliable way to test anything other than MoP Classic at the moment — ports would rely on user testing. PRs welcome on GitHub; issues with reproduction info also welcome.

## Bug reports & feedback

If something looks wrong, please file an issue on [GitHub](https://github.com/Luunae/HideCastBar/issues), even for things like typos. I notice GitHub interactions faster than CurseForge comments. Bug reports of any size are appreciated.

The addon is intentionally scoped to one job — hiding the default Blizzard castbar. Feature requests beyond that scope will probably be declined politely.
